# Plan for review (vignette merge + plot enhancements)

## Hard rules

- No vignette mentions any port-source package by name. Apple-to-apple
  comparisons against port sources live in unit tests
  (`tests/testthat/`) and developer notes
  (`inst/notes/refactor-exceptions.md`); never in `.Rmd`.
- Vignettes use `mfsusie_plot()` and `mfsusie_plot_lfsr()`
  exclusively. `susieR::susie_plot()` is not used in any
  vignette.

## Plot API additions

### `mfsusie_plot()` -- new arguments

| Arg | Values | Default | Effect |
| --- | --- | --- | --- |
| `effect_style` | `"band"`, `"errorbar"` | `"band"` | `"band"`: solid mean line + ribbon (current). `"errorbar"`: per-position dot + vertical bar to credible band. Errors when `credible_bands` empty. |
| `facet_cs` | `"auto"`, `"stack"`, `"overlay"` | `"auto"` | `"overlay"`: all CSes one panel (current). `"stack"`: one row per CS via `layout()`. `"auto"`: stack when `length(cs) >= 3` or affected-region masks pairwise disjoint; overlay otherwise. |
| `lfsr_threshold` | numeric | `0.01` (was `0.05`) | Dashed reference on the lfsr secondary axis. |

### `mfsusie_plot_lfsr()` -- new public function

```
mfsusie_plot_lfsr(fit, lfsr_threshold = 0.01, truth = NULL,
                  cex_max = 6, ...)
```

- No `m` argument. Handles `M = 1` (single panel) and `M > 1`
  (one panel per outcome, tiled).
- Bubble grid: rows = CSes, columns = positions, dot size =
  `-log10(lfsr)` clamped to `[0, cex_max]`.
- Color: `lfsr <= lfsr_threshold` by default; `truth` (length-T
  boolean for M = 1; length-M list for M > 1) overrides for
  vignette use where ground truth is known.
- Errors when `fit$lfsr_curves` empty.

### S3 `plot.mfsusie`

Unchanged: dispatches to `mfsusie_plot(fit, ...)`.

## Data examples (enhance, no new datasets)

### `dnam_example`

Reshaped from `n = 574 / p = 200 / T = 64 / 2 causals` to a
smaller setup matching the methylation case-study vignette:

```
n            = 100
p            = 12
T            = 32
causal_snps  = c(1, 9, 3)         # two on cluster 1 (CpGs 9-16),
                                  # one on cluster 2 (CpGs 25-32)
truth_mask   = list per CS, length-T boolean
description  = "Simulated methylation QTL example. ..."
```

### `rnaseq_example`

Bumped from 1 to 2 causal SNPs at positions 25 and 75. The
per-position effects are smooth random functions on a
power-of-two grid generated by wavelet-coefficient shrinkage,
matching the smooth-function class assumed by the fSuSiE prior.
RNA-seq framing in the description is retained.

```
n            = 574  (susieR::N3finemapping$X[, 1:100])
p            = 100
T            = 128
causal_snps  = c(25, 75)
shape_kind   = "smooth (wavelet-coefficient shrinkage)"
```

## CI / pixi changes

`pixi.toml` gains `r-ggplot2`, `r-cowplot`, `r-reshape2` (under
`[dependencies]` so all environments see them). They are NOT
added to `DESCRIPTION` Suggests. Vignette chunks that use them
gate with `eval = requireNamespace("ggplot2", quietly = TRUE)`
so the package still builds in environments that lack them.
