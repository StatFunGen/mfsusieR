# Implementation tasks

Each numbered group is one PR. Each PR has tests landing in the same
commit per CLAUDE.md. Estimated workload per PR: at most one focused
day; if a task feels larger, split it.

Phase 4 reference tests interleave with Phase 3 implementation: every PR
that adds a numerical routine also adds the matching paired-comparison
test against `mvf.susie.alpha` per the test-fidelity rule
(`~/.claude/projects/-home-gw-GIT-mfsusieR/memory/test_fidelity_and_manuscript_xref.md`).
